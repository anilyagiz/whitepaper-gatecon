# Introduction to Gamecon
 On this platform,where we will bring real players together with the decentralized game world,people will be able to play the games they want according to their taste and interact with people.Players will be able to both have fun and earn money while playing the game in the category they want on the selected blockchain network. The platform aims to be a gateway between blockchain games and gamers. Thanks to the gamification mechanics we have developed, the platform will encourage players to use it for a long time.
